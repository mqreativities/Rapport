Design a modern, clean web and mobile-responsive platform called Rapport — a Nigerian-based counseling and mental wellness platform.

The platform connects verified counselors with clients, organizations, and structured problem-based communities.

Use a minimal, professional UI style (calm colors like soft blue, muted green, white, light grey). Avoid clutter.

⸻

🔹 USER ROLES TO DESIGN FOR

There are 3 main public user roles and 4 admin roles:

Public Roles:
	1.	Client
	2.	Counselor
	3.	Organization Account

Internal Roles:
	4.	Super Admin
	5.	Compliance Admin
	6.	Finance Admin
	7.	Community Oversight Admin
	8.	Support Admin

Design role-based dashboards with different access and permissions.

⸻
